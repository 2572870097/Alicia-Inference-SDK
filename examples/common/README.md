# Common Examples

These scripts are model-agnostic and keep `--model-type` as a parameter.

- `00_quickstart_dummy_policy.py`: no checkpoint required, shows the public API.
- `01_load_and_inspect_policy.py`: load any supported checkpoint and inspect metadata/status.
- `02_synthetic_control_loop.py`: run any supported policy on synthetic inputs.
- `03_manual_policy_lifecycle.py`: lower-level lifecycle demo for any supported policy.
